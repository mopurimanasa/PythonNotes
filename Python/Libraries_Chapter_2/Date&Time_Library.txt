-> The datetime module in Python is used to work with dates, times, and timestamps.
import datetime

or import specific classes:

from datetime import datetime, date, time, timedelta

1. date Class: Represents only the date (year, month, day).

from datetime import date
today = date.today()
print(today)

Output: 2026-07-19

Create a specific date:
d = date(2026, 7, 19)
print(d)

Access attributes:
print(d.year)
print(d.month)
print(d.day)

2. time Class: Represents only time.

from datetime import time
t = time(10, 30, 45)
print(t)
print(t.hour)
print(t.minute)
print(t.second)

Output:
10:30:45
10
30
45

3. datetime Class: Represents both date and time.

from datetime import datetime
now = datetime.now()
print(now)

Output:
2026-07-19 10:30:45.123456

Current date: print(now.date())
Current time: print(now.time())

Access components:
print(now.year)
print(now.month)
print(now.day)
print(now.hour)
print(now.minute)
print(now.second)

4. Formatting Date (strftime): Convert datetime into a formatted string.

from datetime import datetime
now = datetime.now()
print(now.strftime("%d-%m-%Y"))
print(now.strftime("%d/%m/%Y"))
print(now.strftime("%A"))
print(now.strftime("%B"))
print(now.strftime("%I:%M %p"))

Output:
19-07-2026
19/07/2026
Sunday
July
10:30 AM

5. Convert String to Datetime (strptime):

from datetime import datetime
date_string = "19-07-2026"
d = datetime.strptime(date_string, "%d-%m-%Y")
print(d)

Output: 2026-07-19 00:00:00

6. timedelta: Used to add or subtract dates and times:

from datetime import datetime, timedelta
today = datetime.now()
tomorrow = today + timedelta(days=1)
yesterday = today - timedelta(days=1)
print(tomorrow)
print(yesterday)

Add hours:
new_time = today + timedelta(hours=5)

Add minutes:
new_time = today + timedelta(minutes=30)

7. Difference Between Dates:

from datetime import datetime
d1 = datetime(2026, 7, 19)
d2 = datetime(2026, 7, 25)
difference = d2 - d1
print(difference.days)

Output: 6

8. Timestamp:

from datetime import datetime
now = datetime.now()
print(now.timestamp())

Output: 1784457045.123456

9. Replace Date/Time

from datetime import datetime
now = datetime.now()
new = now.replace(year=2030)
print(new)

Interview Example: Find a person's age:

from datetime import date
dob = date(1996, 4, 1)
today = date.today()
age = today.year - dob.year
if (today.month, today.day) < (dob.month, dob.day):
    age -= 1
print(age)
