-> File handling refers to the process of performing operations on a file, such as creating, opening, reading, writing and closing it through a 
programming interface
-> It involves managing the data flow between the program and the file system on the storage device, ensuring that data is handled safely and efficiently.
-> Python provides built-in functions and modules to facilitate file operations such as reading, writing, and manipulating files.

What is the use of File Handling in Python?

1) File Handling is used to store data permanently in a file and retrieve it later.
2) Normally, data stored in variables is lost when the program stops running. File handling allows you to save that data in files such as .txt, .csv, 
.json, .log, etc

Real-Time Example : 

-> Suppose you have a banking application:
name = "Manasa"
balance = 10000
-> If the program closes, these values are lost.
-> Using file handling, you can save them to a file:
with open("account.txt", "w") as file:
    file.write("Manasa,10000")
	
Explaination of above program:
open("account.txt", "w")
-> account.txt → Name of the file.
-> "w" → Write mode -> Creates the file if it does not exist -> Removes all existing content if the file already exists.

as file
-> The opened file is stored in the variable file.

with
-> with automatically closes the file after the block is executed.
-> Now the data remains available even after the program ends.

3) Process Large Files Efficiently Without Using Much Memory

-> Imagine a file containing 10 lakh (1 million) records.
❌ Bad approach:

with open("data.txt", "r") as file:
    data = file.read()  # Loads entire file into memory
	
-> If the file is very large, it consumes a lot of RAM.
✅ Better approach:

with open("data.txt", "r") as file:
    for line in file:
        print(line)
		
-> Here Python reads one line at a time, so memory usage stays low.

4) Automate Tasks Like Reading Configs or Saving Outputs

-> Many applications store settings in configuration files.
config.txt

host=localhost
port=8080

with open("config.txt", "r") as file:
    settings = file.read()
print(settings)

-> Instead of hardcoding values, the program reads them automatically from a file.

Most common file operations are:

1. Open a File:
file = open("account.txt", "r")
-> Opens a file for reading.

2. Read a File: 

Read entire file
with open("account.txt", "r") as file:
    data = file.read()
print(data)

Read one line
with open("account.txt", "r") as file:
    line = file.readline()
print(line)

Read all lines
with open("account.txt", "r") as file:
    lines = file.readlines()
print(lines)

3. Write to a File
with open("account.txt", "w") as file:
    file.write("Manasa,10000")
-> Creates the file if it doesn't exist.
-> Replaces existing content if the file already exists.

4. Append to a File
with open("account.txt", "a") as file:
    file.write("\nRavi,5000")
	
If the file contains:
Manasa,10000

After appending:
Manasa,10000
Ravi,5000

5. Close a File
file = open("account.txt", "r")
# do some work
file.close()
-> Using with automatically closes the file, so close() is usually not needed.

File Modes
Mode	Meaning
"r"		Read
"w"		Write (overwrite)
"a"		Append
"x"		Create file, error if exists
"r+"	Read and write
"w+"	Write and read
"a+"	Append and read

1) Open a File on the Server

-> Assume we have the following file:

demofile.txt
Hello! Welcome to demofile.txt
This file is for testing purposes.
Good Luck!

-> To open the file, use the built-in open() function.
-> The open() function returns a file object, which has a read() method for reading the content of the file

Example:

f = open("demofile.txt")
print(f.read())

-> If the file is located in a different location, you will have to specify the file path, like this:

f = open("D:\\myfiles\welcome.txt")
print(f.read())

Using the with statement:

-> You can also use the with statement when opening a file:
with open("demofile.txt") as f:
  print(f.read())

-> Then you do not have to worry about closing your files, the with statement takes care of that.

2) Close Files:

-> It is a good practice to always close the file when you are done with it.
-> If you are not using the with statement, you must write a close statement in order to close the file:

f = open("demofile.txt")
print(f.readline())
f.close()

3) Read Only Parts of the File

-> By default the read() method returns the whole text, but you can also specify how many characters you want to return:
-> Example:

Return the 5 first characters of the file:
with open("demofile.txt") as f:
  print(f.read(5))
  
3) Read Lines

-> You can return one line by using the readline() method:
with open("demofile.txt") as f:
  print(f.readline())
-> By calling readline() two times, you can read the two first lines:
with open("demofile.txt") as f:
  print(f.readline())
  print(f.readline())
-> By looping through the lines of the file, you can read the whole file, line by line:
with open("demofile.txt") as f:
  for x in f:
    print(x)
	
4) Write to an Existing File

-> To write to an existing file, you must add a parameter to the open() function:

"a" - Append - will append to the end of the file
"w" - Write - will overwrite any existing content

Example:

-> Open the file "demofile.txt" and append content to the file:

with open("demofile.txt", "a") as f:
  f.write("Now the file has more content!")

#open and read the file after the appending:
with open("demofile.txt") as f:
  print(f.read())
  
5) Overwrite Existing Content

-> To overwrite the existing content to the file, use the w parameter:
with open("demofile.txt", "w") as f:
  f.write("Woops! I have deleted the content!")

#open and read the file after the overwriting:
with open("demofile.txt") as f:
  print(f.read())
  
6) Create a New File

-> To create a new file in Python, use the open() method, with one of the following parameters:

"x" - Create - will create a file, returns an error if the file exists
"a" - Append - will create a file if the specified file does not exists
"w" - Write - will create a file if the specified file does not exists

-> Create a new file called "myfile.txt":
f = open("myfile.txt", "x")

7) Delete a File

-> To delete a file, you must import the OS module, and run its os.remove() function:
import os
os.remove("demofile.txt")

Check if File exist:
-> To avoid getting an error, you might want to check if the file exists before you try to delete it:
import os
if os.path.exists("demofile.txt"):
  os.remove("demofile.txt")
else:
  print("The file does not exist")
  
8) Delete Folder

-> To delete an entire folder, use the os.rmdir() method:
import os
os.rmdir("myfolder")

File Types:

1) File with txt Extension

-> A file with a .txt extension is a plain text file used to store text data.

Creating a .txt File
with open("student.txt", "w") as file:
file.write("Manasa")

What happens?
-> student.txt → file name with .txt extension.
-> "w" → write mode.
-> If the file does not exist, Python creates it.
-> "Manasa" is written into the file.

Reading a .txt File:

with open("student.txt", "r") as file:
    data = file.read()
print(data)
Output: Manasa

Appending Data to a .txt File
with open("student.txt", "a") as file:
file.write("\nHyderabad")

Now the file contains:
Manasa
Hyderabad

-> A .txt file is a plain text file used to store textual data. In Python, it is commonly used with file handling operations such as reading 
(r), writing (w), and appending (a) data.

Syntax:
open("filename.txt", "mode")

Examples:
open("data.txt", "r") # Read
open("data.txt", "w") # Write
open("data.txt", "a") # Append

-> .txt is just the file's extension, indicating that the file contains plain text data. Python can create, read, update, and delete such files using 
file handling operations.

3) Changing JSON to Dictionary:

-> To change a JSON to a dictionary, first we import the json module and then we use loads method.

import json
# JSON
person_json = '''{
    "name": "Asabeneh",
    "country": "Finland",
    "city": "Helsinki",
    "skills": ["JavaScrip", "React", "Python"]
}'''
# let's change JSON to dictionary
person_dct = json.loads(person_json)
print(type(person_dct))
print(person_dct)
print(person_dct['name'])

# output
<class 'dict'>
{'name': 'Asabeneh', 'country': 'Finland', 'city': 'Helsinki', 'skills': ['JavaScrip', 'React', 'Python']}
Asabeneh

-> To change a dictionary to a JSON we use dumps method from the json module.

5) Saving as JSON File

We can also save our data as a json file. Let us save it as a json file using the following steps. For writing a json file, we use the json.dump() method, 
it can take dictionary, output file, ensure_ascii and indent.

import json
# python dictionary
person = {
    "name": "Asabeneh",
    "country": "Finland",
    "city": "Helsinki",
    "skills": ["JavaScrip", "React", "Python"]
}
with open('./files/json_example.json', 'w', encoding='utf-8') as f:
    json.dump(person, f, ensure_ascii=False, indent=4)
	
-> In the code above, we use encoding and indentation. Indentation makes the json file easy to read.

6) File with csv Extension

-> CSV stands for comma separated values. CSV is a simple file format used to store tabular data, such as a spreadsheet or database. 
CSV is a very common data format in data science.

-> Example:
"name","country","city","skills"
"Asabeneh","Finland","Helsinki","JavaScript"

import csv
with open('./files/csv_example.csv') as f:
    csv_reader = csv.reader(f, delimiter=',') # we use, reader method to read csv
    line_count = 0
    for row in csv_reader:
        if line_count == 0:
            print(f'Column names are :{", ".join(row)}')
            line_count += 1
        else:
            print(
                f'\t{row[0]} is a teachers. He lives in {row[1]}, {row[2]}.')
            line_count += 1
    print(f'Number of lines:  {line_count}')
	
# output:
Column names are :name, country, city, skills
Number of lines:  1
        Asabeneh is a teacher. He lives in Finland, Helsinki.
Number of lines:  2

9) Binary Files in Python

-> A binary file stores data in the form of bytes (0s and 1s) instead of human-readable text.
-> Examples of binary files:
Images → .jpg, .png
Videos → .mp4
Audio → .mp3
PDF → .pdf
Excel → .xlsx
Word → .docx

-> You cannot easily open and read these files in a text editor because they contain binary data.

Text File vs Binary File

| Text File           | Binary File             |
| ------------------- | ----------------------- |
| Stores characters   | Stores bytes            |
| Human-readable      | Not human-readable      |
| Example: `.txt`     | Example: `.jpg`, `.pdf` |
| Open using `r`, `w` | Open using `rb`, `wb`   |

-> Writing to a Binary File

with open("data.bin", "wb") as file:
    file.write(b"Hello")
	
Here:
wb → write binary mode
b"Hello" → bytes object

-> Reading a Binary File

with open("data.bin", "rb") as file:
    data = file.read()
print(data)
